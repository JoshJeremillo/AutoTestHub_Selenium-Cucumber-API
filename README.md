# AutoTestHub_Selenium-Cucumber-API
Unified automation framework for Web (Selenium + Cucumber + Java) and API (Postman) testing


# Simple Automation

This repository contains sample automation using:

- Selenium + Cucumber + Java
- API tests using Postman

## Setup

1. Clone the repo:

git clone <repo-url>

2. Install dependencies:

mvn clean install

3. Run Cucumber tests:

mvn test

4. Import Postman collection from `postman/SimpleAPI.postman_collection.json`
